# YouTube 頻道資料分析系統

本專案為一個以 Python 開發的 YouTube 頻道資料分析系統，透過自動化爬蟲擷取頻道與影片資料，並進行整理與長期趨勢分析，最終以圖表與網頁方式呈現結果。

此專案整合資料擷取 資料儲存 與資料視覺化流程，展示從資料蒐集到分析應用的完整實作能力。

---

## 專案負責內容

爬蟲設計與資料擷取流程
資料庫設計與資料儲存
後端邏輯開發
資料分析與視覺化實作
Web 介面建置

---

## 專案動機

一般使用 YouTube 時，僅能觀察單一影片或短期數據，難以分析頻道長期表現與成長趨勢。

本專案希望透過資料分析方式

建立自動化資料收集系統
分析影片觀看數與喜歡數變化
觀察頻道長期成長趨勢
提供視覺化分析結果

---

## 使用技術

Python
Selenium
Flask
SQL Server
matplotlib

---

## 系統流程

使用者輸入頻道名稱
系統透過 Selenium 搜尋 YouTube
擷取頻道與影片資料
資料儲存至資料庫
進行資料整理與分析
生成趨勢圖
透過 Flask 網頁顯示結果

---

## 成果展示與分析

本系統針對 YouTube 頻道進行長期資料分析，涵蓋 2014 至 2023 共 10 年資料，並依據每一年分別產生

每年觀看數折線圖
每年喜歡數折線圖

共計產出 20 張圖表，完整呈現頻道長期經營成果與內容表現。

---

## 圖表產出說明

系統會依據影片發布日期，自動將資料依年份分類，並針對每一年生成

觀看數趨勢圖
喜歡數趨勢圖

每張圖以時間為橫軸 數據為縱軸，呈現該年度所有影片的表現變化。

## 整體分析結論

透過 2014 至 2023 年共 20 張圖表，可觀察頻道發展歷程

初期 數據較低且穩定
中期 開始出現波動與成長
後期 高流量影片增加
最終 進入穩定成長階段

觀看數與喜歡數呈現正相關
高觀看影片通常伴隨高互動
顯示內容品質對頻道成長具有關鍵影響
